== Description ==
The plugin helps you to create the subscription plan for your products.

[FREE VERSION DEMO LINK](http://subscriptionfree.phoeniixxdemo.com/wp-admin/) | [FREE VERSION DOCUMENTATION LINK](http://hemani.codiixx.com/wp-content/uploads/2018/01/Subscription.pdf)

= Features: =

* The plugin is compatible with our <strong> Support Ticket System by Phoeniixx plugin </strong>
* Option to create the subscription plan for a product based on either the Days or Weeks.
* Option to enable / disable the subscription. 
* Option to set the subscription length.
* Can enable / disable the payment , overdue and suspend option.
* Option to set the number of days to send email before the payment due date.
* Option to set the number of days to send email after payment due date.
* Option to set the number of days to suspend the subscription service if payment is not received.
* Email reminder option.
* In Admin Panel the full subscription details will be displayed.
* Option to edit any order in the admin panel.

== Installation ==

1. Install [WordPress](https://wordpress.org/)
2. Go to your WordPress admin panel, and then to Plugins. Click on "Add New" and then upload the zip file of the plugin using the "Upload Plugin" button you can find on top of the screen.

== Screenshots ==

1. Displays the general settings from the backend.
2. Displays the subscription settings in a product.
3. It shows the subscription details of a customer.
4. It shows the subscribe button on a product page.
